Agent Distribution v1.5.12 - frontend background preload

Changes:
- Keeps v1.5.11 backend speed/warmup optimization.
- Frontend now avoids duplicate dashboard API calls by caching loaded tab data and reusing in-flight requests.
- After Dashboard loads successfully, Admin UI preloads other allowed tabs in the background.
- After Sales Adviser Dashboard loads successfully, Agent UI preloads Team, Products, Orders, Reward and Withdrawals in the background.
- Background preload errors do not interrupt the current page.
- When user later opens a preloaded tab, data should appear faster because it is already in frontend memory.

Notes:
- Full frontend cache still refreshes when user clicks Refresh or uses table search/pagination.
- Backend remains based on v1.5.11.
